# Pet Halo

Pet Halo is an independent, unofficial macOS companion that aims to display a transparent, always-on-top usage halo near Codex Pet. It does not modify Codex Desktop or Codex Pet.

> Pet Halo is an independent, unofficial project for Codex. It is not affiliated with or endorsed by OpenAI.

## Status

The project is in **M0 protocol feasibility**. This milestone only validates whether the locally installed experimental Codex app-server protocol can safely provide:

- five-hour and weekly rate-limit availability;
- account usage availability;
- loaded thread metadata and live context-token usage;
- a reliable or user-driven target-thread selection strategy.

No Halo UI is implemented in M0. Protocol results must come from schemas generated by the current local Codex CLI; the project does not read or depend on Codex internal SQLite databases.

## M0 commands

```sh
python3 -m unittest discover -s Tests -p 'test_*.py'
python3 Tools/ProtocolProbe/probe.py --transport stdio --output-dir /tmp/pet-halo-probe
```
