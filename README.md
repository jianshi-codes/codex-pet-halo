# Pet Halo

Pet Halo is an independent, unofficial macOS companion that aims to display a transparent, always-on-top usage halo near Codex Pet. It does not modify Codex Desktop or Codex Pet.

> Pet Halo is an independent, unofficial project for Codex. It is not affiliated with or endorsed by OpenAI.

## Status

**M1 application skeleton is implemented locally; smoke testing and Draft PR CI are the remaining gate checks.** The native Swift 6 application runs as a menu-bar/accessory shell with no Dock icon or normal window. It contains only lifecycle, version, placeholder menu status, and Quit behavior. It does not connect to Codex or read Usage data.

M0 protocol feasibility remains complete with **PASS-CORE / PARTIAL-OPTIONALS**. The validated future MVP uses capability-driven data:

- weekly rate-limit remaining percentage and reset time as the primary halo;
- the Halo-owned app-server connection state;
- account usage summary and daily usage in an expanded view;
- a five-hour rate-limit segment only when a 300-minute window is returned;
- a Context segment only when a supported shared thread source and target selection are available.

Missing optional capabilities are hidden or shown as unavailable and are never estimated. No Halo overlay or Usage UI is implemented in M1. Protocol results must come from schemas generated by the current local Codex CLI; the project does not read or depend on Codex internal SQLite databases.

## Development commands

```sh
make bootstrap
make generate
make build
make test
make m0-tests
make check
```

XcodeGen 2.46.0 is required. `project.yml` is the editable project source of truth; the generated Xcode project is committed and must not be hand-edited.
