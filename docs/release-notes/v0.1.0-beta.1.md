# Pet Halo 0.1.0 Beta 1

Pet Halo is an independent, unofficial macOS companion for Codex. This first planned Public Beta presents weekly remaining Usage, an optional five-hour window, and Today compared with historical daily peak around a visible Codex Pet, with safe window/free-floating fallbacks.

## Highlights

- Original, non-activating Pet Ring and menu-bar experience.
- Pet-first following with calibrated Codex-window and free-floating fallbacks.
- Light/dark, Increase Contrast, Differentiate Without Color, Reduce Transparency, and Reduce Motion support.
- Read-only, fail-closed Codex CLI compatibility with no Usage persistence.

## Requirements

- macOS 14.0 or later on Apple silicon or Intel.
- Supported Codex CLI and Desktop versions listed in `docs/COMPATIBILITY.md`.
- Accessibility permission only when following is enabled.

## Beta note

This is prerelease software. Report compatibility problems through the sanitized issue form. Never attach raw protocol payloads, tokens, account identity, conversation content, private screenshots, raw Accessibility errors, executable paths, or PIDs.
