# Pet Halo 0.1.0 Beta 1 — Unsigned Developer Preview

Pet Halo is an independent, unofficial macOS companion for Codex. This Source Public Beta presents weekly remaining Usage, an optional five-hour window, and Today compared with historical daily peak around a visible Codex Pet, with safe window/free-floating fallbacks.

> **Unsigned preview warning:** The downloadable artifact is unsigned and is not notarized by Apple. macOS may block its first launch. Building from the reviewed source remains available. Do not treat this artifact as a signed or notarized release, and do not override Gatekeeper unless you have independently verified the GitHub source, release checksum, and repository provenance.

## Highlights

- Original, non-activating Pet Ring and menu-bar experience.
- Pet-first following with calibrated Codex-window and free-floating fallbacks.
- Light/dark, Increase Contrast, Differentiate Without Color, Reduce Transparency, and Reduce Motion support.
- Read-only, fail-closed Codex CLI compatibility with no Usage persistence.

## Requirements

- macOS 14.0 or later on Apple silicon or Intel.
- Supported Codex CLI and Desktop versions listed in `docs/COMPATIBILITY.md`.
- Accessibility permission only when following is enabled.
