# Pet Halo 0.1.0 Beta 2 — Unsigned Developer Preview

Pet Halo is an independent, unofficial macOS companion for Codex. Beta 2 adds the verified Weekly reset date to the compact Pet Ring capsule while preserving the existing Usage model, target hierarchy, and fallback surfaces.

> **Unsigned preview warning:** The prepared artifact is unsigned and is not notarized by Apple. macOS may block its first launch. Do not treat it as a signed or notarized release, and do not override Gatekeeper unless you have independently verified the GitHub source, release checksum, and repository provenance.

## Highlights

- The Weekly capsule now displays its reset date when Codex provides one, for example `W 39% · Jul 27`.
- The reset timestamp is rendered in the user's local timezone, while accessibility retains the full local date, time, and timezone context.
- No Usage semantics, remaining-percentage semantics, Codex protocol requests, or protocol compatibility rules changed.
- The exact supported Codex CLI and Desktop versions remain unchanged from Beta 1.

## Requirements

- macOS 14.0 or later on Apple silicon or Intel.
- Supported Codex CLI and Desktop versions listed in `docs/COMPATIBILITY.md`.
- Accessibility permission only when following is enabled.

This is prerelease software. The Beta 2 release identity is prepared but not published by this change. The unsigned Universal artifact name is `Pet-Halo-0.1.0-beta.2-unsigned-universal.zip`; it is not notarized by Apple.
