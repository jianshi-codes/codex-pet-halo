# Pet Halo 0.1.0 Beta 2 — Unsigned Developer Preview

Pet Halo is an independent, unofficial macOS companion for Codex. Beta 2 adds the verified Weekly reset date to the compact Pet Ring capsule while preserving the existing Usage model, target hierarchy, and fallback surfaces.

> **Unsigned preview warning:** The downloadable artifact is unsigned and is not notarized by Apple. macOS may block its first launch. Do not treat it as a signed or notarized release, and do not override Gatekeeper unless you have independently verified the GitHub source, release checksum, and repository provenance.

## Highlights

- The Weekly capsule now displays its reset date when Codex provides one, for example `W 39% · Jul 27`.
- The reset timestamp is rendered in the user's local timezone, while accessibility retains the full local date, time, and timezone context.
- Newer pre-1.0 Codex CLI versions at or above `0.145.0-alpha.18` may run provisionally after required read-only runtime capabilities succeed; the exact reviewed registry remains unchanged.
- Malformed, too-old, explicitly denied, 1.x, and runtime-incompatible CLI versions still fail closed. Optional five-hour and Today data remain capability-gated and are never estimated.
- No Usage semantics, remaining-percentage semantics, protocol request scope, Pet Ring visuals, Weekly reset-date behavior, Pet discovery, Accessibility targeting, or reviewed Codex Desktop versions changed.

## Requirements

- macOS 14.0 or later on Apple silicon or Intel.
- Supported Codex CLI and Desktop versions listed in `docs/COMPATIBILITY.md`.
- Accessibility permission only when following is enabled.

## Beta note

This is Beta software. The unsigned Universal artifact is `Pet-Halo-0.1.0-beta.2-unsigned-universal.zip`. It is not signed with a Developer ID certificate and is not notarized by Apple.
